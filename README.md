# FoxNibbles Landing Page

Premium landing page for **FoxNibbles** (roasted makhana / fox nuts).

## How it works
- `index.html` uses **Tailwind CSS via CDN** (no build step).
- Product/founder images live in `public/`.

## Deploy on GitHub Pages
1. Create a GitHub repo from this folder.
2. Commit these files (at minimum):
   - `index.html`
   - `public/` (all images)
   - `README.md`
3. In your repo: **Settings → Pages**
4. Set **Build and deployment** to:
   - **Deploy from a branch**
   - Branch: `main` (or your default branch)
   - Folder: `/ (root)`
5. Save. Your site will be available at the GitHub Pages URL.
